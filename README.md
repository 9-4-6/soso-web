# soso-web
知识库系统

## 项目启动

```sh
pnpm install
```

### 项目编译启动

```sh
pnpm dev
```

### 访问

```sh
http://localhost:3008/
```

## 技术栈
* vue 3.5.17
* ts
* pina 3.0.3
* route 4.5.1
* element-plus 2.10.4
* vite 7.0.0
* node v20.19.4
